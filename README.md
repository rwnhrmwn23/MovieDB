## üõ† Prerequisites

Before running the project, ensure you have the following set up:

*   **Kotlin Version**: `2.3.0`
*   **Java Version**: `21` (Source & Target Compatibility)
*   **Android Studio**: Compatible with the above versions (Ladybug or newer recommended).

### üîë API Key Configuration

This project requires a TMDB API Key to fetch data.

1.  Get your API Key from **[TMDB Settings > API Read Access Token](https://www.themoviedb.org/settings/api)**.
2.  Create or open the `local.properties` file in the root directory.
3.  Add the following line with your key:

```properties
TMDB_API_KEY=YOUR_OWN_IS_HERE
```
> **Note**: Do not share your `local.properties` or commit it to version control if it contains secrets.

## üèó Architecture

This app follows **Clean Architecture** principles to ensure separation of concerns, testability, and scalability.

### Layers & Folder Structure

| Layer | Folder | Description |
| :--- | :--- | :--- |
| **App** | `com.onedev.moviedb` | Root package. |
| **Core** | `core/` | Common utilities, extension functions, DI modules, Network config, and Theme. |
| **Data** | `data/` | Implementation of Repositories, API interactions (Ktor), Paging Sources, and DTO Mappers. |
| **Domain** | `domain/` | **Pure Kotlin** layer containing business logic (Use Cases), Models, and Repository interfaces. |
| **Presentation** | `presentation/` | UI Logic (ViewModels) and Screen Composable functions using Jetpack Compose. |
| **UI** | `ui/` | Theme definitions (Color, Type, Shape) generated by Android Studio. |

## ‚úÖ Features

*   **Genre List**: Browse movies by official genres.
*   **Discover Movies**: Filter and view movies by selected genres.
*   **Movie Detail**: View comprehensive movie info (poster, backdrop, synopsis, rating).
*   **Reviews**: Read user reviews.
*   **Trailers**: Watch YouTube trailers directly in the app.
*   **Endless Scrolling**: Seamless browsing with Paging 3 integration.
*   **Error Handling**: Robust handling of network errors and empty states.

## üì° API Endpoints

The application interacts with the following TMDB API endpoints:

| Feature | HTTP Method | Endpoint | Description |
| :--- | :--- | :--- | :--- |
| **Genres** | `GET` | `/3/genre/movie/list` | Get the list of official genres for movies. |
| **Discover Movies** | `GET` | `/3/discover/movie` | Discover movies, filtered by genres. |
| **Movie Search** | `GET` | `/3/search/movie` | Search for movies by title. |
| **Movie Details** | `GET` | `/3/movie/{movie_id}` | Get the top level details of a movie. |
| **Movie Reviews** | `GET` | `/3/movie/{movie_id}/reviews` | Get the user reviews for a movie. |
| **Movie Videos** | `GET` | `/3/movie/{movie_id}/videos` | Get the videos (trailers) for a movie. |
| **Images** | `GET` | `https://image.tmdb.org/t/p/w500/{path}` | Load movie posters and backdrops. |
